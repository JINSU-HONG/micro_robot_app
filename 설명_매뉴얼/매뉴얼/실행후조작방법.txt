프로그램 실행 후 터미널 콘솔에 키보드를 통해 값을 넣고 엔터를 입력하여 조작을 한다.


<모드 변경>
1 + enter : INIT mode로 변경 - 통신 및 인버터 보드의 상태가 정상인지 확인하는 모드
2 + enter : INV_TEST mode로 변경 - 인버터를 테스트하여 정상인지 확인하는 모드
3 + enter : IDLE mode로 변경 - 전류를 0으로 제어하는 모드
4 + enter : RUN mode로 변경 - 주어진 자기장과 힘 지령을 수행하는 모드
0 + enter : 프로그램을 종료한다.

(최근 프로그램 업데이트로 2, 3 모드를 제거 함)
(2+enter, 3+enter 두 가지 입력을 4+enter 과 같게 RUN mode로 변경되도록 함)
(WEBCAM을 사용하는 경우 webcam의 화면상에서 0+enter 을 하여도 프로그램이 종료 된다.)


<지령 변경>
INIT mode에서 자기장 지령이나 힘 지령을 변경 시키면 바로 실행되지 않는다.
RUN mode에 있어야 설정된 지령이 수행된다.
(RUN mode에서 자기장 지령이나 힘 지령을 변경하면 바로 변경되어 수행한다.)

b + enter : 수행할 수 있는 자기장 지령의 종류를 표시한다.
            나타난 보기의 번호와 enter를 입력하면 수행 자기장 지령이 변경된다.
f + enter : 수행할 수 있는 힘 지령의 종류를 표시한다.
            나타난 보기의 번호와 enter를 입력하면 수행 힘 지령이 변경된다.

(4코일 시스템의 경우 x축에 정렬된 자석에는 y축 힘을 인가할 수 없고, y축에 정렬된 자석에는 x축 힘을 인가할 수 없어 자기장과 힘 지령이 동시에 수행될 수 있도록 지령을 만들어 둠 - 자기장과 힘 지령이 한 번에 수정됨 (b+enter와 f+enter의 기능이 같음)


<기타 조작>
n + enter : 제어할 전류(인버터 보드)의 수 / 개수를 입력 후 enter (중대형시스템 : 8, 소형시스템 : 4)
m + enter : 전류 제한 방법 변경 / proportional limit을 사용하는 경우 이동 방향이나 정렬 방향에 왜곡이 없다.
w + enter : 자기장이나 힘 지령이 진동하는 함수일 경우 해당 함수의 주파수 변경 / 원하는 주파수를 float 형으로 입력한뒤 enter
o + enter : 통신이 중간에 끊기는 경우 n번 이전의 전류 지령을 측정 전류로 대신하여 사용 / n을 입력 후 enter
l + enter : 마이크로로봇의 현재 위치를 0, 0으로 수정


<카메라 조작> - opencv 설치후 variables.h의 USE_WEBCAM이 1로 설정되어 있어야 함, 콘솔이 아닌  출력 화면 위에 입력
0 : 프로그램 종료 (webcam 뿐만이 아닌 전체 프로그램)
esc : webcam 동작만 종료
클릭 : 4코일 시스템에서 자기장, 힘 지령이 mouse input 인 경우 클릭한 지점으로 이동하는 위치 제어 수행
c : 마이크로로봇의 위치 센싱에 사용되는 offset 위치를 재설정. 마이크로로봇을 멈춰놓은 뒤 c를 누르고 수 초간 기다려야한다.
